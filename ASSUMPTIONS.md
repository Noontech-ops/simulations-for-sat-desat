# Assumptions

| Assumption | Value |
|------------|--------|
| Adsorbent | Al-Fumarate (Al-Fum) |
| Model | pyIAST |
| CH4 isotherms | Experimental |
| N2 isotherms | Experimental |
| CO2 isotherm | Experimental |
| O2 isotherm | Surrogate |
| O2 uptake | 0.85 × N2 uptake |
| O2 sensitivity bounds | 0.7×–1.0× N2 |
| Adsorbent framework | Rigid |
| Pressure | 1 bar |
| CH4/N2 feed | 50/50 mol% |
| CO2/N2 feed | 15/85 mol% |

O2 = 0.85 × N2 because O2 and N2 have similar kinetic diameters, interact weakly with Al-Fum through physisorption, and therefore exhibit adsorption behavior of the same order. The 0.85 factor represents a midpoint between bounded sensitivity assumptions of 0.7× and 1.0× N2 uptake.
